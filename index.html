Real Satellite Data
Physical risk variables use ISRO Bhuvan flood susceptibility layers, MODIS 8-day Land Surface Temperature composites, and Sentinel-2 NDWI water indices — government-verified, tamper-proof remote sensing data validated against NRSC flood atlas 1998–2022.
🔢
PCA-Derived Weights
Layer weights (0.45/0.30/0.25) are not hand-picked. They are derived from Principal Component Analysis on a 25×12 normalised ward matrix using scikit-learn. PC1 explains 82.6% of variance — statistically confirming that a single composite index is valid.
📊
Confidence-Adjusted Scoring
Every variable carries a confidence score (0–1) derived from data quality × source reliability × resolution fit. Low-confidence variables (e.g. IRDAI insurance proxy at 0.45) contribute proportionally less. Model mean confidence: 0.709 — clearly disclosed, not hidden.
🎲
Monte Carlo Uncertainty
1,000 simulations per ward with Gaussian noise scaled to data confidence. Average CI width: 3–4 CRCS points. Borderline wards (CI crossing band boundary) are flagged for manual underwriter review — the model does not pretend to precision it doesn't have.
